About Yuzu Companion

A personal AI partner built with simplicity in mind

Why I Built This

I wanted something simple, ready-to-go, and personal. No complicated setups, no subscription fees - just a companion that works out of the box.

Inspired by SillyTavern and other AI chat platforms, but stripped down to the essentials.

Originally for personal use only... but who knows? Maybe someone else will find it useful too.

The Tech Stack

  • Python & Flask - Simple web framework
  • SQLite - Lightweight database
  • Ollama - Local AI models
  • OpenRouter - Cloud AI fallback
  • Pico.css - Minimal, semantic CSS

Built with tools that don't require a PhD to understand.

Design Philosophy

  • Simplicity first - No unnecessary complexity
  • Mobile-friendly - Chat from anywhere
  • Personal touch - It feels like your companion

Built on the shoulders of digital giants.

Built with love, code, and probably too many late nights.
