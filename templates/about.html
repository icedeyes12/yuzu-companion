About Yuzu Companion

Yuzu Companion

A personal AI partner built with simplicity in mind

Why I Built This

I wanted something simple, ready-to-go, and personal. No complicated setups, no subscription fees - just a companion that works out of the box.

The Tech Stack

  • Python - Simple web framework backend
  • Flask - Lightweight web framework
  • SQLite - Lightweight database
  • Ollama - Local AI models
  • OpenRouter - Cloud AI fallback
  • HTML/CSS/JS - Clean web UI with Pico.css

Design Philosophy

  • Simplicity first
  • Mobile-friendly
  • Personal touch
