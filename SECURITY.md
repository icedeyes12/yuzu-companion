## Reporting a Vulnerability

If you discover a security vulnerability, **do not** open a public GitHub issue.

Instead, please report it responsibly:

1. **Email**: Send details to the repository owner directly
2. **Private Security Advisory**: Use GitHub's [Private Vulnerability Reporting](https://github.com/icedeyes12/yuzu-companion/security/advisories/new) feature

When reporting, please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We aim to respond within 48 hours and will work with you to coordinate a fix and disclosure.
